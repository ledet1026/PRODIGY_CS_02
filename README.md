# PRODIGY_CS_02
#  Image Encryption Tool (Pixel Manipulation)

This project is a simple **image encryption and decryption tool** developed using Python.   It demonstrates basic **pixel manipulation techniques** by applying mathematical operations (XOR) on image pixel values.


The tool allows users to:
- Encrypt images using a numeric key
- Decrypt images using the same key
- Save the processed image as a new file

---

##  Internship Task Requirement

This project was developed as part of the **ProDigy Infotech Internship Task-02**:

> Develop a simple image encryption tool using pixel manipulation. You can perform operations like swapping pixel values or applying a basic mathematical operation to each pixel. Allow users to encrypt and decrypt images.

---

## Features

-  Load images from file path
-  Encrypt image using XOR operation
-  Decrypt image using same key
-  Save output images automatically
-  Input validation and error handling
-  Simple command-line interface (CLI)

---

##  Technology Used

- Python 3
- PIL (Pillow) – image processing
- NumPy – pixel manipulation
- OS module – file handling
